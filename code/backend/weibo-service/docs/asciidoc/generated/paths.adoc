==== delete one user
....
GET /user/delById
....


===== 说明
Delete one user info specified by userId


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a"]
|===
|类型|名称|说明|类型
|**Query**|**userId** +
__可选__|The user ID of a WeiBo user,should be a Long Integer|integer (int64)
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|**200**|OK|object
|**204**|No Content|无内容
|**401**|Unauthorized|无内容
|**403**|Forbidden|无内容
|===
