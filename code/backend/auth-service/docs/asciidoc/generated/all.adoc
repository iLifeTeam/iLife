[[_authusingpost_7]]
==== User log in
....
POST /auth/auth
....


===== 说明
Auth one iLife user by giving account.password


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|**200**|OK|object
|**204**|No Content|无内容
|**401**|Unauthorized|无内容
|**403**|Forbidden|无内容
|**501**|user not exists|无内容
|**502**|account and password not match|无内容
|===


===== 消耗

* `application/json`


===== 生成

* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/auth/auth
----


===== HTTP响应示例

====== 响应 200
[source,json]
----
"object"
----
